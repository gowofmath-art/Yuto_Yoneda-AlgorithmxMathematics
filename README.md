# 📐 Yuto Yoneda: Algorithm × Mathematics Solutions & Insights

> **「本書是我在演算法與資料結構領域的啟蒙之作。它讓我看見，程式碼不是憑空產生的規則，而是數學結構在計算機世界的實體投影。」**

本專案是我深入研讀米田優俊（Yuto Yoneda）名著——**《「演算法×數學」全彩圖解學習全指南：從基礎開始，一次學會24種必學演算法與背後的關鍵數學知識及應用》**（原著：問題解決のための「アルゴリズム×数学」が基礎からしっかり身につく本）的完整實踐筆記與節末習題解答庫。

This repository logs my comprehensive solutions and mathematical insights for Yuto Yoneda's renowned book, *"Algorithm × Mathematics"*. It serves as my foundational gateway into the world of computer science algorithms, built upon the bedrock of rigorous mathematical reasoning.

---

## 💡 為什麼是這本書？ (Why This Book?)

對於擁有數學背景（Mathematical Background）的我而言，這本書對「數學思考」與「第一性原理（First Principles）」的追求極其友善。它不教人盲目地套用程式套件，而是從最根本的數論、線性代數與機率幾何出發，推導出經典演算法的演進脈絡。

這本書徹底點燃了我對演算法架構與複雜度最佳化的熱情。在這裡，我拒絕快速複製貼上，堅持每一題都做到：
1. **數理白箱化**：用嚴謹的數理語言與 $\mathcal{O}$ 符號，寫下最純粹的邏輯與數學語言和程式碼。
2. **底層純手刻**：捨棄高級內建函數，從迴圈、指標與邏輯閘出發，手刻出運算邏輯。
