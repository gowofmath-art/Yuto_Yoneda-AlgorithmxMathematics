### 🛠️ 問題 2.1.4：進位制工程轉換（核心亮點：拒絕內建套件，底層純手刻）
本題捨棄 Python 內建的 `bin()` 函數 ，完全採用**第一性原理的短除法**進行乘冪迴圈實作 。

#### 1. 二進位轉十進位 (`twototen`)
利用連續模十（`% 10`）抽取出二進位的每一位數 $k_i$，再透過乘冪累積：
$$X = \sum_{i=0}^{n} k_i \cdot 2^i$$

#### 2. 十進位轉 $N$ 進位通用演算法 (`tentotwo` / `tentothree`)
利用基底（Base）的整除與模運算性質。當十進位數 $n > 0$ 時：
* 餘數：
* $$a = n \pmod{\text{Base}}$$
* 更新商數：
* $$n = n \mathbin{//} \text{Base}$$
* 位數對齊：利用 $10^{\text{count}}$ 將取得的餘數逆序組裝成新型態數字。

此手刻演算法精準地模擬了數學上的離散短除法結構，成功通過二進位與三進位的雙向驗證！
