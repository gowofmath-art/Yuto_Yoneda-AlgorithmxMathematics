## 習題核心解構與工程洞察

### 問題 2.3.1 & 2.3.2：基礎函數值計算與特殊函數邊界
* **數學事實**：
  - 多項式函數：$f(x) = x^3 \implies f(1)=1, \, f(5)=125, \, f(10)=1000$。
  - 對數與無理數指數：$\log_2 8 = 3$ 且 $100^{1.5} = 100^{\frac{3}{2}} = 1000$。
* **高斯符號與無條件進位（地板與天花板函數）**：
  - 數學定義中，高斯符號 $\lfloor x \rfloor$ 代表不大於 $x$ 的最大整數（Floor），而 $\lceil x \rceil$ 代表不小於 $x$ 的最小整數（Ceiling）。
  - **工程實作細節**：
    - $\lfloor 2.4 \rfloor = 2$ 且 $\lfloor -2.4 \rfloor = -3$
    - $\lceil 2.4 \rceil = 3$ 且 $\lceil -2.4 \rceil = -2$
  - **工程洞察**：在 Python 實作中，直接使用整數強制轉型 `int(-2.4)` 會輸出 `-2`（此為向零捨入，Truncation toward zero），而非數學定義上的 $\lfloor -2.4 \rfloor = -3$。因此，在處理包含負數的資料清洗或時間序列離散化時，必須嚴格調用 `math.floor()` 與 `math.ceil()`，以確保邊界條件符合數學模型的預期。

### 問題 2.3.3：跨尺度函數圖形的幾何視覺化
為了直觀觀察不同成長速率函數的幾何特性，本題利用 `numpy` 在指定定義域內生成連續稠密點，並使用 `matplotlib` 進行多維度圖形繪製：
* **線性函數 ($y = 2x + 3$)**：成長速率為常數（$f'(x) = 2$），圖形呈直線。
* **指數函數 ($y = 10^x$)**：成長速率隨 $x$ 呈爆炸性增長，在資料科學中常用於描述未受限的群體成長或演算法的最壞複雜度（如 $O(2^n)$）。
* **對數函數 ($y = \log_4 x$ 與 $y = \frac{\log_2 x}{2}$)**：
  - **幾何發現**：經由換底公式推導：
    $$\log_4 x = \frac{\log_2 x}{\log_2 4} = \frac{\log_2 x}{2}$$
  - **實證結果**：兩函數在代數上完全等價。於畫布上實測繪圖，兩條函數軌跡完美重合，從幾何視角印證了對數換底公式的正確性。

## 嚴謹數學推導：一般化值域變換與單調性保持

在本章的延伸思考中，探討了如何將一個已知的嚴格單調遞增函數，透過線性轉換完美映射至標準化區間 $(0, 1)$，並確保其核心數學性質不發生退化。

### 定理陳述
設一函數 $g(x)$ 於定義域內滿足：
1. **嚴格單調遞增**：若 $x_1 < x_2$，則 $g(x_1) < g(x_2)$。
2. **值域限制**：其值域為開區間 $(A, B)$，即 $A < g(x) < B$，其中 $B > A$。

試證明：新構造之函數 $$f(x) = \frac{g(x) - A}{B - A}$$ 其值域必為 $(0, 1)$，且同樣保持嚴格單調遞增。

### 核心邏輯證明

#### 1. 值域（Value Range）的映射證明
由定理前提可知，對於定義域內的任意 $x$，其函數值 $g(x)$ 受到嚴格上下界約束：
$$A < g(x) < B$$

由於我們欲構造 $f(x)$ 的分子，將不等式各邊同時減去常數 $A$（不等號方向保持不變）：
$$0 < g(x) - A < B - A$$

已知值域上界大於下界（$B > A$），故兩者之差必為正實數，即 $(B - A) > 0$。
將不等式各邊同時除以該正數 $(B - A)$，不等號方向依然保持不變：
$$\frac{0}{B - A} < \frac{g(x) - A}{B - A} < \frac{B - A}{B - A}$$

簡化代數式後得到：
$$0 < f(x) < 1$$

**得證：新函數 $f(x)$ 的值域被完美限縮於開區間 $(0, 1)$ 之中。**

#### 2. 嚴格單調遞增性（Strict Monotonicity）的保持證明
設定義域內任意給定兩點 $x_1, x_2$，且滿足大小關係 $x_1 < x_2$。
根據前提條件，$g(x)$ 為嚴格單調遞增函數，由其數學定義直接推得：
$$g(x_1) < g(x_2)$$

對該不等式兩邊同時減去相同的實數常數 $A$，不等式依然成立：
$$g(x_1) - A < g(x_2) - A$$

接著，兩邊同時除以正數 $(B - A)$。根據序理論（Order Theory），正數的乘除法具有保序性（Order-preserving）：
$$\frac{g(x_1) - A}{B - A} < \frac{g(x_2) - A}{B - A}$$

由此可得新函數值的關係：
$$f(x_1) < f(x_2)$$

**得證：當 $x_1 < x_2$ 時，必導出 $f(x_1) < f(x_2)$，函數 $f(x)$ 完美保持了嚴格單調遞增的幾何特性。**

## 本章學習與大數據思維洞察

2.3 節的精髓在於**「離散與連續的交織」**以及**「特徵縮放（Feature Scaling）的數學本質」**：
1. **高斯符號的演算法應用**：高斯符號是連接連續實數軸與離散計算機記憶體（如分桶、雜湊、時序離散化）的橋樑。理解其在負數區間的邊界特徵，能避免大數據清洗時產生致命的 $1$ 誤差（Off-by-one error）。  

2. **機器學習特徵標準化的數學根基**：最後推導的一般化公式 $f(x) = \frac{g(x) - A}{B - A}$，本質上正是機器學習（ML）中極為關鍵的 **Min-Max Scaling（最大最小標準化）**。透過這次嚴謹的代數證明，我們在底層邏輯上確立了：**無論原始特徵數據的分佈如何（只要其映射關係是嚴格單調的），特徵縮放不僅能將數據安全地限制在 $(0, 1)$ 的標準空間內，更絕對不會破壞數據原本的序關係（Order）與相對大小特徵**。這種數學上的穩定性，是機器學習優化演算法（如梯度下降法）能夠快速收斂的根本基石。
