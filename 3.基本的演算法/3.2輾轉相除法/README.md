# 📐 Chapter 3.2 輾轉相除法

本篇章研讀米田優俊《演算法×數學》第 3.2 節，本章節探討如何求兩正整數A,B的最大公因數，如果使用最單純的做法，會很花時間，時間複雜度來到O(n)，可是如果利用輾轉相除法去優化，時間複雜度可以來到O(log(A+B))

以下為各小節：

3.2.1 單純的演算法
講述最簡單求最大公因數的做法：就是從 1到min(A,B) 中所有的正整數去一一驗證，若有找到新的數字就更新。    

3.2.2 有效率的演算法：輾轉相除法  
講述使用輾轉相除法，使用wlile迴圈判斷 每一次A,B誰大誰小，較大者更新為 大%小，直到不能除為止。  
時間複雜度來到 $O(\log(A+B))$
