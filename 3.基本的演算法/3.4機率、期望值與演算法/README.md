# 📐 Chapter 3.4 機率、期望值與演算法  

本篇章研讀米田優俊《演算法×數學》第 3.4 節，此章節前半部分將介紹機率和期望值的基礎、期望值的線性關係，對於後續的3.5章節-蒙地卡羅法相當重要，後半段將介紹利用這些期望值的觀念去改善演算法的3個例子。  

以下為各小節： 

3.4.1 機率是什麼？  
簡述機率的定義，這邊是以「古典機率」作為說明。  

3.4.2 期望值是什麼？  
期望值的數學定義：某個試驗，共有N個個事件，並對應機率與報酬，期望值就是各別機率與報酬的乘機總和。
$$E(X) = \sum_{i=1}^{N} x_i \cdot p_i$$

符號詳細說明：
$E(X)$：表示隨機變數 $X$ 的期望值（Expected Value）。  
$\sum_{i=1}^{N}$：求和符號（Sigma），表示將編號 $i = 1$ 到 $N$ 的所有項目加總。  
$x_i$：第 $i$ 個事件所對應的報酬（或數值）。  
$p_i$：第 $i$ 個事件發生的機率，且滿足所有機率總和為 1（即 $\sum p_i = 1$）。  

3.4.3 期望值的線性關係是什麼？  
若有兩個隨機變數 $X$ 與 $Y$，以及任意常數 $a, b, c$，則期望值滿足以下線性公式：  
$$E(aX + bY + c) = aE(X) + bE(Y) + c$$  
「和的期望值」即為「期望值之和」  

3.4.4 應用例(1)：2個骰子  
問題：投擲兩個公正的骰子，以等機率出現上面的點數$A_i$ , $B_i$，當同時丟擲兩個骰子，點數和的期望值是多少？  
最單純的作法是對全部組合去做計算，然而需要計算 $6 \times 6 = 36$ 種情況，再取平均。  
實際上利用期望值的線性性質 $E(X + Y) = E(X) + E(Y)$  
計算第一個骰子的點數期望值再加上第二個骰子的點數期望值，也是答案，而且好算非常多。


3.4.5 應用例(2)：2個骰子的一般化  
同上，將上面的問題更加一般化，投擲兩個正N面骰子，各點數分別為 $B_i$ , $R_i$。  
求同時投擲兩個骰子的點數期望值。  

解法與上一題一致。  

3.4.6 應用例(3)：再選擇題的考試中隨機地回答  
有一份考試共有 $N$ 題選擇題。
對於第 $i$ 題（其中 $1 \le i \le N$）：  
共有 $P_i$ 個選項（其中只有一個是正確答案）。  
該題的配分為 $Q_i$ 分。  
太郎因為完全不會寫，決定每一題都隨機亂猜。請問太郎整份考試得分的期望值是多少？

整份考試的總得分期望值公式為：

$$E(X) = \sum_{i=1}^{N} \frac{Q_i}{P_i}$$

絕不要全搜尋的做法。
