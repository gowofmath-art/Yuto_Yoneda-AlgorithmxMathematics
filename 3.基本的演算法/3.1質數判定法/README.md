# 📐 Chapter 3.1 質數判定法

本篇章研讀米田優俊《演算法×數學》第 3.1 節，本章節探討如何判定質數的問題，雖然上在一章節最後已經提到。但上一章主要是期望我們自己用數學方式練習，本章節開始都用數學的思考加上演算法的實作。  

以下是各小節：  
3.1.1 單純的質數判定法  
講解最單純的質數判定法，用一個個比N小的數字去除，雖然直觀，但是時間複雜度為 $O(n)$  

3.1.2 快速的質數判定法  
實際上不用這麼慢，若要找N是否為質數，只需要調查到 $\sqrt{N}$ 的數字即可，時間複雜度可降為 $O(\sqrt{N})$  

3.1.3 反證法是什麼？  
為了說明上一小節說的快速質數判定法，這邊提及「反證法」：假設事實P是錯誤的話，會導致矛盾，進而推導 ~P 才是正確的。是個很棒的說明。一般程式的書未必會這樣講。  

3.1.4 證明演算法的正確性
利用剛說的演算法，證明「快速的質數判定法」是正確的。  

3.1.5 應用例：因數列舉  
最後利用質數判定法去說明列舉某數N的所有因數。時間複雜度達到 $O(\sqrt{N})$
